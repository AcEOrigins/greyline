2. **User Enters Credentials** ‚Üí Email/password validation
3. **Database Authentication** ‚Üí Validates against `users` table
4. **Session Creation** ‚Üí Stores user data in localStorage
5. **Role Check** ‚Üí Determines admin vs employee access
6. **Redirect** ‚Üí Main interface with role-based UI
